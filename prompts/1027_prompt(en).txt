# INSTRUCTION
Analyze the user's sentence and extract structured information following the [JSON SCHEMA] below.

- Each field must strictly follow the data type and rules defined in the schema.
- If a specific piece of information is missing in the sentence, set its value to `null`.

- `activity`: List all activities the user intends to do, expressed as nominalized forms (ending with “-기” in Korean).
- `activity_tags`: Select one or more from the [Google Maps Place Tags] list below that best correspond to the user's activity.
  These tags will be used directly as search keywords for the map API.
- `time`: Extract both specific time information (`specific`) and the estimated duration (`length_hour`).
- `companion`: Extract the companion type (`type`, e.g., "friend", "family", "coworker") and the number of people (`num` as a number).
- `budget`: Extract the budget type (`type`: "per_person" or "total") and the amount (`amount` in numeric string).
- The output must contain **only JSON** — do not include explanations, markdown syntax, or code fences.
